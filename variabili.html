In questo esempio vediamo i tipi di variabili nel linguaggio Javascript

Javascript e' un linguaggio tipizzato, ma non in forma esplicita.

Cio' significa che non si deve specificare il tipo della variabile durante la dichiarazione, ma verra' asseganto un tipo nel momento in cui inizializziamo la variabile.

La keyword per dichiarare una variabile e' var

La keyword per dichiarare una costante e' const

Premendo F12 si apre la console degli strumenti del browser.

console.log consente di stampare valori in console del browser.
